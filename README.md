# Innovathens map

A simple interactive map with hubs, moke, incubators etc. Built for [Innovathens](http://www.innovathens.gr/).

## Libraries used

* [Leaflet](http://leafletjs.com/)
* [Leaflet.Control.FullScreen](https://github.com/brunob/leaflet.fullscreen/)
